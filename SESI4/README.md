# Scope tugas: 
Membuat program kalkulator sederhana dengan memanfaatkan function, struktur switch-case, serta memanggil function dari file JavaScript lain menggukanan mekanisme module (module.exports dan require).

# Tujuan pembelajaran:
- Memahami cara membuat function di JavaScript
- Menerapkan konsep modular program (export & import antar file)
- Menggunakan switch-case untuk membuat menu kalkulator
- Mengelola data menggunakan array sederhana
- Menjalankan beberapa file JS secara terstruktur menggunakan Node.Js
